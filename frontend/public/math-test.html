BİLGİN Math Rendering Test

This page tests the math rendering functionality that should be working in the BİLGİN chat application.

Test 1: Inline Math
The Pythagorean theorem states that $x^2 + y^2 = r^2$ for a right triangle.
Test 2: Display Math - Summation
The sum of the first n natural numbers is: $$\sum_{i=1}^{n} x_i = \frac{n(n+1)}{2}$$
Test 3: Fractions
A simple fraction: $\frac{1}{2}$ and a complex fraction: $\frac{a+b}{c+d}$
Test 4: Greek Letters
Greek letters are commonly used in mathematics: $\alpha, \beta, \gamma, \delta, \pi, \sigma, \omega$
Test 5: Complex Integral
The Gaussian integral: $$\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}$$
Test 6: Matrix
A 2x2 matrix: $$\begin{pmatrix} a & b \\ c & d \end{pmatrix}$$
Test 7: Limits
The definition of a derivative: $$\lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$
